# 最优配置详情

> A 股与美股各自的最优永久组合配置及改进路径

## A 股最优（2014-2025）

### 配置参数

```yaml
assets:
  - ticker: 510300.SS  # 沪深300 ETF
    weight: 0.25
  - ticker: 511010.SS  # 国债 ETF
    weight: 0.25
  - ticker: 518880.SS  # 黄金 ETF
    weight: 0.25
  - ticker: 511990.SS  # 货币基金
    weight: 0.25
params:
  rebalance_freq: never
  rebalance_threshold: 0.07
  stop_loss: 0.10
  stop_loss_mode: crisis
  crisis_weights:
    510300.SS: 0.03
    511010.SS: 0.39
    518880.SS: 0.34
    511990.SS: 0.24
```

### 指标对比

| 指标 | 月度再平衡（基线） | + 阈值 7% | + 危机模式 | 沪深300 买入持有 |
|------|:-----------:|:---------:|:---------:|:------------:|
| Sortino | 0.724 | 0.809 | **0.996** | 0.313 |
| 年化收益 | +6.12% | +6.70% | **+7.09%** | +7.98% |
| 最大回撤 | -12.46% | -11.64% | **-10.69%** | -44.99% |
| Calmar | 0.491 | 0.576 | **0.663** | 0.177 |
| 最终资产 | 1,987,902 | 2,118,413 | **2,208,234** | 2,429,246 |

### 改进路径

```
0.724  月度再平衡（基线）
  → 0.809  阈值 7% 替代月度（+12%）
    → 0.996  + 危机模式 10%（+23%）
```

年化收益从 +6.12% 提升到 +7.09%，回撤从 -12.46% 降到 -10.69%。

## 美股最优（2005-2025）

### 配置参数

```yaml
assets:
  - ticker: QQQ   # 纳斯达克 100
    weight: 0.40
  - ticker: AGG   # 综合债 ETF
    weight: 0.20
  - ticker: GLD   # 黄金 ETF
    weight: 0.20
  - ticker: SHY   # 短期国债 ETF
    weight: 0.20
params:
  rebalance_freq: never
  rebalance_threshold: 0.10
```

### 指标对比

| 指标 | 30% QQQ + 阈值 7% | 40% QQQ + 阈值 10% | SPY 买入持有 |
|------|:-----------------:|:-----------------:|:------------:|
| Sortino | 0.831 | **0.862** | 0.428 |
| 年化收益 | +8.98% | **+10.28%** | +10.56% |
| 最大回撤 | -19.48% | -23.15% | **-54.35%** |
| Calmar | 0.461 | **0.444** | 0.194 |
| 最终资产 | $5,394,859 | **$7,771,587** | $7,904,876 |

### 关键差异

- **美股不需要危机模式**：QQQ 跌后反弹快，砍仓反而错过反弹
- **美股可以更激进**：40% QQQ 比传统的 30% 多捕获了上行趋势
- **收益接近 SPY，回撤仅为其 43%**：年化 +10.28% vs SPY +10.56%，回撤 -23.15% vs -54.35%
